# Archive

This directory marks superseded or exploratory material that is retained for
provenance but excluded from the Chapter 1 submission claim.

Archive candidates include:

- orientation-only v1 analyses;
- dated one-off workflow definitions after their artifacts are frozen;
- exploratory RF, involucre, mucilage, hair, MaxEnt and palaeoclimate branches;
- figures or tables generated before the final completeness rules were fixed.

Files are not moved automatically in the Phase 1 PR because mass renaming would
break historical workflow paths and make review harder. A file may be moved here
only after the submission bundle records its original path, commit and checksum.
